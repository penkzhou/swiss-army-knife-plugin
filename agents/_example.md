# Example Agent

This is a template for creating subagents. Subagents are specialized Claude instances that handle specific tasks.

## Creating a New Agent

1. Create a new `.md` file in this directory
2. Define the agent's role and capabilities in frontmatter
3. Provide detailed instructions for how the agent should behave

## Agent Guidelines

- Be specific about the agent's expertise and limitations
- List the tools the agent needs access to
- Provide examples of expected input/output
- Define success criteria for the agent's tasks

---

Delete this file once you've created your first real agent.
